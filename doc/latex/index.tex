\chapter{Demo univers}
\hypertarget{index}{}\label{index}\index{Demo univers@{Demo univers}}
Voici la documentation de Demo \doxylink{class_univers}{Univers}. Demo \doxylink{class_univers}{Univers} est un programme conçu pour simuler le mouvement de particules dans un univers. Il existe 3 types de forces qui peuvent affecter une particule \+: la force d\textquotesingle{}interaction du potentiel de Lennard-\/\+Jones, la force d\textquotesingle{}interaction gravitationnelle et la force du potentiel gravitationnel.

De plus, il existe 3 types de conditions limites \+:


\begin{DoxyEnumerate}
\item Réflexion \+: Les particules subissent une force de réflexion lorsqu\textquotesingle{}elles se rapprochent à une distance inférieure à r\+Cut\+Reflexion de l\textquotesingle{}une des limites de l\textquotesingle{}univers.
\item Absorption \+: Les particules disparaissent lorsqu\textquotesingle{}elles atteignent les limites de l\textquotesingle{}univers.
\item Périodique \+: Les particules se déplacent dans un univers périodique.
\end{DoxyEnumerate}

D\textquotesingle{}autre part, pour simplifier les calculs, les forces d\textquotesingle{}interaction ne sont calculées que si la distance entre les particules est inférieure à r\+Cut. Ainsi, seules les particules qui contribuent à une attraction significative sont prises en compte.\hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Utilisation du programme}{Utilisation du programme}}\label{index_autotoc_md0}
Le programme peut être exécuté avec les options suivantes. Si une valeur n\textquotesingle{}est pas fournie, les valeurs par défaut sont utilisées \+:


\begin{DoxyItemize}
\item {\ttfamily -\/h, -\/-\/help} \+: Afficher ce message d\textquotesingle{}aide.
\item {\ttfamily -\/v, -\/-\/version} \+: Afficher les informations de version.
\item {\ttfamily -\/-\/ldX \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 0) \+: Définir la longueur en X.
\item {\ttfamily -\/-\/ldY \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 0) \+: Définir la longueur en Y.
\item {\ttfamily -\/-\/ldZ \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 0) \+: Définir la longueur en Z.
\item {\ttfamily -\/-\/r\+Cut \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 2.\+5) \+: Définir la distance de coupure.
\item {\ttfamily -\/-\/condition\+Limite \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 2) \+: Définir la condition limite.
\item {\ttfamily -\/-\/r\+Cut\+Reflexion \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 2\texorpdfstring{$^\wedge$}{\string^}(1/6)) \+: Définir la distance de coupure pour la réflexion.
\item {\ttfamily -\/-\/limiter\+Vitesse} (valeur par défaut \+: faux) \+: Limiter la vitesse.
\item {\ttfamily -\/-\/energie\+Desiree \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 0.\+005) \+: Définir l\textquotesingle{}énergie désirée.
\item {\ttfamily -\/-\/force\+LJ} (valeur par défaut \+: faux) \+: Utiliser la force de Lennard-\/\+Jones.
\item {\ttfamily -\/-\/force\+IG} (valeur par défaut \+: faux) \+: Utiliser la force gravitationnelle.
\item {\ttfamily -\/-\/force\+PG} (valeur par défaut \+: faux) \+: Utiliser le potentiel gravitationnel.
\item {\ttfamily -\/-\/epsilon \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 5.\+0) \+: Définir la valeur d\textquotesingle{}epsilon.
\item {\ttfamily -\/-\/sigma \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 1.\+0) \+: Définir la valeur de sigma.
\item {\ttfamily -\/-\/G \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: -\/12) \+: Définir la valeur de G.
\item {\ttfamily -\/-\/delta \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 0.\+00005) \+: Définir la valeur de delta.
\item {\ttfamily -\/-\/t\+Final \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} (valeur par défaut \+: 19.\+5) \+: Définir la valeur de t\+Final.
\item {\ttfamily -\/-\/adresse\+Fichier \texorpdfstring{$<$}{<}valeur\texorpdfstring{$>$}{>}} \+: Définir l\textquotesingle{}adresse du fichier d\textquotesingle{}entrée.
\end{DoxyItemize}

Si aucune variable n\textquotesingle{}est fournie, le programme exécutera les tests de performance sur les différents types de collections et mesurera le temps pour différentes tailles de données.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Exemple d\textquotesingle{}utilisation \+:}{Exemple d\textquotesingle{}utilisation \+:}}\label{index_autotoc_md1}
{\ttfamily ./demo\+\_\+univers -\/-\/ldX 250 -\/-\/ldY 130 -\/-\/force\+LJ -\/-\/adresse\+Fichier lecture.\+vtu}

Cela initialisera la simulation avec un univers de dimensions (250 x 130), avec une distance de coupure de 2,5 et une condition limite de type absorbant. Pour le calcul de la force de Lennard-\/\+Jones, epsilon est égal à 5 et sigma est égal à 1. La valeur de delta est de 0,00005 et le temps final de la simulation est de 19,5.
